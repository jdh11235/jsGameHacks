jsGameHacks
git.io/jsgamehacks

mods/hacks/improvements for HTML5/JavaScript games

The following scripts do fun stuff to various games.

To install a script: drag its button to your bookmarks bar.
To run a script, click that bookmark while playing the game that it modifies.
(because the scripts are designed to automatically update, they don't work without internet connection.)

NOTE: Not all web browsers are created equal!
These scripts use advanced web standards and will only work correctly if you're using a recent version of Chrome or Firefox.

2048 • http://git.io/2048

Lets you change the numbers on tiles! 2048/EditMode
Downloads a save file of your current game. 2048/ExportSave
Imports a 2048 save file (overwriting your current game). 2048/ImportSave

Cookie Clicker • http://orteil.dashnet.org/cookieclicker/

Enter numbers into box, get cookies. Click again to remove box. CookieClicker/GiveBox

GameAboutSquares • http://gameaboutsquares.com

Change your highest unlocked level. GameAboutSquares/UnlockLevel
