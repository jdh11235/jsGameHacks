jsGameHacks
git.io/jsgamehacks

mods/hacks/improvements for HTML5/JavaScript games

The following scripts each change various games in some way.

To install a script: drag its button to your bookmarks bar.

To install a script: copy its code, create any bookmark for a placeholder, edit the bookmark, and replace the bookmark's URL with that code.

To run a script, click that bookmark while playing the game that it modifies.

NOTE: Some scripts might not work in web browsers that are really old, mobile, or just plain weird.
If you're having problems, try using a modern browser like Chrome or Firefox
